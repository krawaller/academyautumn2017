Start of object: STARTOBJEKT!
End of object: SLUTOBJEKT!

Start of object: STARTATTRIBUT!
End of object: SLUTATTRIBUT!

Start of child object: STARTBARNOBJEKT!
End of child object: SLUTBARNOBJEKTT!

Colon: KOLON!
Comma: SEPARERA!

Object names should start with an upper case and then alternating upper and lower case
Attributes should end with upper case letter while all other letters are lower case
Values are middle letter upper case while all other letters are small. If there are no single middle letter then the two in the middle are both upper case
Child objects are all lower case letters as children have a hard time differentiating between lower and upper
For semantic reasons whitespace means nothing, Jon Snow.

This is a "new line" separated language
